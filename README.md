
# 🍽️ Zomato Restaurant Data Analysis Using Python

## 📌 Project Overview
This project analyzes the Zomato Bangalore restaurant dataset 
to understand what factors influence restaurant ratings and 
overall performance. The analysis covers 45,000+ restaurants 
across 93 locations in Bangalore, India.

## 🎯 Objective
To analyze restaurant data and identify patterns between 
restaurant attributes (cost, online ordering, table booking, 
location, cuisine) and customer ratings using Python.

## 📊 Dataset
- **Source:** Kaggle
- **Dataset:** Zomato Bangalore Restaurants
- **Rows:** 45,000+
- **Columns:** 17

## 💡 Key Insights
- Average restaurant rating in Bangalore is **3.72**
- Restaurants with **online ordering** have higher ratings
- Restaurants with **table booking** have significantly higher ratings
- **North Indian and Chinese** are most popular cuisines
- **BTM and Koramangala** have the most restaurants
- **Sankey Road** is the most expensive location
- Most restaurants fall in **Budget or Mid Range** category
